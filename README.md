# 图书馆管理系统（Vue + Spring Boot）

基于 `Vue3 + Element Plus + Spring Boot + MyBatis-Plus + MySQL` 开发的图书馆管理系统，支持管理员与读者双角色登录，涵盖图书管理、借阅管理、读者管理、分类管理、预约管理、罚款管理、个人信息与密码修改等功能。

---

## 项目简介

本项目是一个面向图书馆业务场景的前后端分离管理系统，适合作为数据库课程设计、Java Web 课程设计、毕业设计或综合实训项目展示。

系统已完成管理员端与读者端的界面划分，并对页面风格、表格结构、表单弹窗、统计看板等进行了统一优化，整体更加现代化和完整。

## 六、目前前端还在继续优化中的点

虽然已经很完整了，但有些功能还要看后端接口是否完全匹配，例如：

- 预约数据是否真实联通
- 罚款数据是否完全接后端
- 分类接口是否全部可用
- 统计图是否接真实数据库统计

也就是说：

前端结构和页面已经基本到位，部分新增模块还需要后端接口配合才能完全闭环。
